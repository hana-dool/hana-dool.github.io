> ## Python Docstring Generator 

-  docstring 을 자동으로 생성시켜주는 Extension 입니다.

![png](/assets/images/Python/57_1.png)

```python
def custom(x,y) :
  return x+y
```

- 위처럼 그냥 custom 함수를 저장했다고 합시다. 

```python
def custom(x,y) :
    """[summary]

    Args:
        x ([type]): [description]
        y ([type]): [description]

    Returns:
        [type]: [description]
    """
    return x+y
```

- 그 이후에는 위처럼 , 쌍따움표를 딱 써주면 자동으로 함수설명의 템플릿을 생성해줍니다.
