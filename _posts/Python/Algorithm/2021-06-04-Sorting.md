# <center><font size="15"> 선택 정렬 </font></center>

- 선택 정렬이란, 다음과 같은 순서로 이루어진다.
  - 주어진 리스트중에서 최소값을 찾는다. 
  - 그 값을 맨 앞에 위치한 값과 교체한다. 
  - 이를 계속 뒤에까지 계속한다. 

| 패스 |        테이블         | 최솟값 |
| :--: | :-------------------: | :----: |
|  0   | [**9,1,6,8,4,3,2,0**] |   0    |
|  1   | [0,**1,6,8,4,3,2,9**] |   1    |
|  2   | [0,1,**6,8,4,3,2,9**] |   2    |
|  3   | [0,1,2,**8,4,3,6,9**] |   3    |
|  4   | [0,1,2,3,**4,8,6,9**] |   4    |
|  5   | [0,1,2,3,4,**8,6,9**] |   6    |
|  6   | [0,1,2,3,4,6,**8,9**] |   8    |

- 구현

```python
def selection_sort(arr):
    for i in range(len(arr) - 1):                   # 어디서부터 탐색할 것인지.
        min_idx = i
        for j in range(i+1, len(arr)):              # i 부터 탐색하면, i+1 부터 탐색
            if arr[j] < arr[min_idx]:               
                min_idx = j                          
        arr[i], arr[min_idx] = arr[min_idx], arr[i] # 값을 Swap
    return arr
```

# <center><font size="15"> 거품 정렬 </font></center>

- 두 인접한 원소를 검사하여 정렬하는 방법

![png](/assets/images/Py_Algorithm/2_2.png)

```python
def bubble_sort(arr):
    for i in range(len(arr)-1,0,-1):
        for j in range(i):
            if arr[j] > arr[j+1]:
                arr[j],arr[j+1] = arr[j+1],arr[j]
```
