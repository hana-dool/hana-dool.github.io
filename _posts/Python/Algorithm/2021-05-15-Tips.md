### 데이터 Sapping

- 다음과 같이 하게 되면, a,b 값이 바뀌게 된다.

```python
a = 1 , b = 2
a,b = b,a #a=2,b=1 이됨
```

- 위와같이 하게 된다면 a 와  b 의 값이 바뀌게된다. 

```
a[i],a[j] = a[j],a[i] # 옆과 같이 list 에도 적용할 수 있다. 
```

## 갯수가 정해지지 않은 Input

- input 의 갯수가 정해지지 않고 계속 들어온다면 언제 멈추어야 할까? 

```python
while True : 
    a = input()
    if not a :
        break
```
