# DIV

DIV(a,b) : a 를 b 로 나누는 연산의 몫만 출력

- DIV(11,2) = 5



# Max , Min

MAX(a,b) : a,b 중 큰 값 반환

MIN(a,b) : a,b 중 작은값 반환



# Round

- ROUND(num,[decimals]) : 숫자를 지정한 자릿수로 반올림
  - ROUND(3.5,1) = 4

# ZN

- ZN(expression) : Null 이 아니면 식을 반환, Null 이면 0을 반환
